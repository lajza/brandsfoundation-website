# Brands Foundation — Website

Official website of Brands Foundation, a non-profit organization building strong brands for Silesian companies.

**Live site:** [brandsfoundation.pl](https://brandsfoundation.pl)

## Tech

Single-file HTML/CSS/JS website. No frameworks, no build step.

- Custom font: Silesianum (8 weights)
- Canvas-based animated beam hero
- Fully responsive

## Structure

```
index.html        — entire website
fonts/            — Silesianum OTF files
brand/            — logos and photos
partners/         — partner logos
```
